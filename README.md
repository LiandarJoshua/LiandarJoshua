## Featured Project

### [Engineering Review Platform](https://github.com/LiandarJoshua/EngineeringReviewingTool)

Autonomous multi-agent code review system — posts inline GitHub PR comments, tracks developer progress over time, and runs scheduled repo health scans.

- 11-stage **LangGraph** DAG pipeline (security, architecture, testing, scalability, tech debt)
- **RAG knowledge base** backed by OWASP Top 10, clean architecture principles, DDIA
- GitHub automation: PR review bot, commit status checks, `/review` comment commands
- Stack: FastAPI · Celery · ChromaDB · Neo4j · Ollama · React · Docker Compose
